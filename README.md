# Messenger
## Описание

Приложение позволяет:
- участвовать в переписке с контактами 
- добавлять/удалять пользователей
- прикреплять фото/видео, файл, геолокацию

Дизайн приложения - [Mockup](https://www.figma.com/file/Zrwzoirt0PzHxl8IliXiAZ/Messenger?node-id=0%3A1)

Текущая версия приложения - [Deployed app](https://hiderr-messenger.herokuapp.com/)

## Установка

- `npm install` — установка стабильной версии,
- `npm start` — сборка проекта + запуск версии для разработчика.
- `npm run test` — запуск тестов.
